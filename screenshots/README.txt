Screenshot Directory

This directory will store screenshots taken during gameplay.

To take a screenshot while playing the game, press F12.

The blog post references the following screenshots:
1. menu.png - A screenshot of the main menu
2. gameplay.png - A screenshot of actual gameplay
3. level_complete.png - A screenshot showing level completion

You can rename your screenshots to match these filenames for the blog post,
or update the blog post to reference your actual screenshot filenames.
